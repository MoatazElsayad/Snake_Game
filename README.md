# Sophisticated Snake Game

A modern, interactive Snake game built with Python and Pygame.

## Features
- **Sophisticated Frontend**: Clean UI with Dark and Light mode themes.
- **Customizable**: Choose your snake color and enter your username.
- **Difficulty Levels**: Easy, Medium, and Hard modes with different speeds and win conditions.
- **Score Tracking**: Real-time score and length display.
- **Solid Backend**: Object-oriented design with separated concerns.

## Requirements
- Python 3.x
- Pygame

## Installation

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the game:
   ```bash
   python main.py
   ```

## Controls
- **Arrow Keys** or **WASD**: Move the snake.
- **Mouse**: Interact with the menu (Select Color, Difficulty, Theme).
